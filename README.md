# jquery.sortme

## Usage

Given a table with a structure of: 
``` html
<table>
    <thead>
        <tr>
            <th>Click Me!</th>
        </tr>
    </thead>
    <tbody>
      <tr>
        <td>Sortable Values!</td>
      </tr>
    </tbody>
</table>
```
Initialize the plugin on your table like so:
``` javascript
$('table').sortme();
```
